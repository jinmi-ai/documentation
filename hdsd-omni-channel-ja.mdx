# **チャットボットモジュール利用ガイド - Jinmi Omnichannel**

## **パート1：アカウントの登録/ログイン**

### **1. アカウント登録**

ステップ1：登録画面で → **_Googleで続行_** を選択

ステップ2：Googleアカウントでログイン

ステップ3：店舗情報を設定 → **_続行_** をクリック

これでJinmi Omnichannel への登録が完了しました。

### **2. ログイン**

Omniプラットフォームへのログイン方法は2つあります。

- <u>方法1：アカウントでログイン</u>

ステップ1：リンクにアクセス：[<u>Jinmi Portal</u>](https://portal.jinmi.ai/auth?return_url=https%3A%2F%2Fportal.jinmi.ai%2Fapi%2Foauth%2Fauthorize%3Fclient_id%3Domnichannel_client_id%26redirect_uri%3Dhttps%253A%252F%252Fomni.jinmi.ai%252Fportal%252Fsso%252Fcallback%26response_type%3Dcode%26scope%3Dopenid%2Bprofile%2Bemail%26state%3DLw%253D%253D)

ステップ2：登録済みのメールアドレスと有効なパスワードを入力し、「ログイン」をクリック

\

- <u>方法2：Googleでログイン</u>

ステップ1：ログイン画面で「Googleでログイン」をクリック

ステップ2：ログインしたいアカウントを選択

これでJinmi Omnichannel システムへのログインが完了しました。

## **パート2：LINEをOmnichannelに接続する**

### **1. LINE公式アカウントを作成する**

ステップ1：**_Create a LINE Official Account_** を選択

ステップ2：必要な情報を入力 → **_Continue_** を選択

ステップ3：情報を確認 → **_Submit_** を選択

ステップ4：**_Application Complete_** が表示されたら → **_Go to LINE Official Account Manager_** を選択

### **2. Messaging APIを有効にする**

ステップ1：メイン画面にWelcomeが表示されます => **_Next_** を選択 => 次に **_Home Screen_** を選択してホームページに移動します。

ステップ2：**_Settings_** を選択 → **_Messaging API_** を選択

ステップ3：**_Enable Messaging API_** をクリック

ステップ3：**_Account Name_** を入力 → **_Agree_** を選択

ステップ4：必要な情報を入力 → **_OK_** を選択

### **3. 接続情報を取得する**

ステップ1：**_LINE Developers Console_** にアクセス

ステップ2：**_Provider_** 画面で接続チャネルを選択

ステップ3：チャネル接続画面で **_Messaging API_** を選択

ステップ3：下にスクロールして **_Issue_** を選択

### **4. LINEをJinmiに接続する**

ステップ1：**_Jinmi Omnichannel_** にアクセス → **_Settings_** → **_Inboxes_** を選択

ステップ2：**Add Inbox** を選択

**ステップ3：** **_LINE_** を選択し、チャネル情報を入力

**_LINE_** 画面に戻り → **_Channel ID、Channel Secret、Token_** をそれぞれコピーしてチャネル情報（LINE Channel ID、LINE Channel Secret、LINE Channel Token）に入力

ステップ3：**_Create LINE Channel_** を選択

ステップ4：**_Agents_**（スタッフ）を追加 → **_Add Agents_** を選択

ステップ5：Jinmiシステムから **_Webhook URL_** をコピー

ステップ6：LINEに戻り → **_Webhook URL_** を貼り付け → **_Save_** を選択

ステップ7：**_Settings_** → **_Response settings_** を選択 → **_Chat、Webhook_** を有効にする

ステップ8：LINEの接続が完了しました。Omnichannelに戻り → **_More Settings_** を選択

これでLINEのOmnichannelへの接続が完了し、使用できるようになりました。エージェントはLINE経由でお客様とチャットできます。

**パート3：AIチャットボットの作成とトレーニング方法**

### **1：チャットボットを作成する**

ステップ1：**_Jinmi Agent_** にアクセス → **_Create a New Assistant_** を選択

ステップ2：以下の情報を入力：

1. チャットボット名を入力（_例：商品相談、よくある質問への回答、24時間カスタマーサポート用チャットボット_）
2. チャットボットの役割を説明
3. メイン商品（_例：Jinmi chatbot AI_）
4. ニーズに合わせて適切なボット機能を選択
5. 解決メッセージの送信時間を設定
6. **_Create_** をクリック

### **2. チャットボットをトレーニングする**

- FAQを作成：よくある質問と事前設定された回答のリスト。

チャットボットはこれらの質問と回答のペアを使用してお客様に自動的に応答し、エージェントの時間を節約します。

ステップ1：チャットボット作成後 => **_FAQS_** をクリック => **_新しいFAQを作成_** をクリック

ステップ2：FAQを追加：

1. お客様がよく聞く質問を入力（_例：この商品に保証はありますか？_）
2. 回答を入力
3. この質問を使用するチャットボットを選択
4. 画像を追加（必要な場合）
5. **_Create_** をクリック

- チャットボットにドキュメントを添付する

Jinmiドキュメントは、チャットボットが読んで学習するための情報、記事、ガイドを保存する場所です。

チャットボットはこれらのドキュメントを使用して、より正確にお客様に回答します。

ステップ1：**_Jinmi Agent_** で => **_Documents_** を選択

\

ステップ2：**_Create a new document_** を選択

ステップ3：情報を入力

1. ドキュメント名を入力（_例：Omni利用ガイド_）
2. ドキュメントの種類を選択
3. ドキュメントのリンクを貼り付け
4. チャットボットを選択
5. **_保存_** をクリック

### **3. チャットボットをチャネルに接続する**

ステップ1：**_Jinmi Agent_** を選択 → LINEに接続したいチャットボットの(...)アイコンをクリック → **_Connect Inbox_** を選択

ステップ2：**_Connect a new inbox_** を選択

ステップ3：接続したいチャネルを選択（Facebook/Zalo/Website/WhatsApp/Instagram…）

ステップ4：**_Create_** をクリックして受信トレイの接続を完了

この時点で、チャットボットは以下のことができます：

- お客様に自動的に応答する
- ドキュメントに基づいてコンテンツを理解する
- 多言語をサポートする

**注意：** チャットボットのドキュメントは1つの言語で作成するだけで十分です。お客様が別の言語でメッセージを送信した場合でも、チャットボットは理解し、お客様の言語で返信します。

**4. チャットボットをテストする**

- 方法1：Jinmi Omnichannelで直接チャットボットをテスト

ステップ1：**_Jinmi Agents_** にアクセス → テストしたいチャットボットの(...)アイコンをクリック → **_Edit Assistant_** を選択

\
**	**ステップ2：**_Playground_** を選択 → メッセージを入力してチャットボットの応答をテスト

- 方法2：お客様としてチャットする：個人アカウントを使用 → ページ/LINEにメッセージを送信

トレーニングされたコンテンツに基づいてチャットボットが自動的に応答するのを確認できます。ボットが正しく回答しない場合は、FAQやドキュメントを追加できます。

**パート4：まとめ**

これで、Jinmi Omnichannelでのチャットボットの作成、トレーニング、デプロイが完了しました。

チャットボットは自動的にお客様への相談対応と返信をサポートし、時間の節約とカスタマーサポートの効率化に役立ちます。
