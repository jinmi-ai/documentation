Jinmi AIには、始めるための2つの強力な製品があります：AI駆動のコンテンツ制作のための**Content Hub**と、あらゆるチャネルでの顧客会話管理のための**オムニチャネル**です。どちらも無料プランで利用可能 — クレジットカードは不要です。

## アカウントを作成

[portal.jinmi.ai](https://portal.jinmi.ai/)にアクセスして無料アカウントを登録してください。1分以内にセットアップが完了し、すぐにご利用いただけます。

<Note>
  無料プランには月200メッセージ、トレーニングファイル3件、統合受信トレイ1件が含まれています。開始にクレジットカードは不要です。
</Note>

## 製品を選択

Jinmi AIの2つのコア製品は、ビジネスの異なるニーズに対応しています。当面の目標に合った製品を選ぶか、両方をご利用ください。

<CardGroup cols={2}>
  <Card title="Content Hub" icon="wand-magic-sparkles" href="/content-hub/overview">
    1つのアイデアから数分でブログ、画像、動画、スクリプトを作成。マーケター、ショップオーナー、高品質なコンテンツを継続的に必要とするすべての方に。
  </Card>

  <Card title="オムニチャネル" icon="comments" href="/omnichannel/overview">
    Facebook、Zalo、ウェブサイトなどからのメッセージを1つの受信トレイに統合。AIチャットボットで24時間365日、あなたが眠っている間も自動で相談・受注対応。
  </Card>
</CardGroup>

## 運用手順

<Steps>
  <Step title="portal.jinmi.aiで登録">
    [**portal.jinmi.ai**](http://portal.jinmi.ai)にアクセスして無料アカウントを作成。クレジットカードは不要です。
  </Step>
  <Step title="開始する製品を選択">
    ログイン後、ダッシュボードから**Content Hub**または**オムニチャネル**を選択。いつでも製品間の切り替えが可能です。
  </Step>
  <Step title="最初のチャネルを接続（オムニチャネル）または最初のアセットを作成（Content Hub）">
    - **オムニチャネル**: 最初の受信トレイを接続 — Facebook、Zalo OA、またはウェブサイトのライブチャット。無料プランでは統合受信トレイ1件をサポート。
    - **Content Hub**: コンテンツタイプ（ブログ記事、SNSキャプション、商品説明）を選び、トピックやアイデアを入力。AIが数秒で完全な下書きを生成します。
  </Step>
  <Step title="AIをトレーニング（任意ですが推奨）">
    トレーニングファイルを最大3件アップロード — 商品カタログ、ブランドガイドライン、またはFAQドキュメント — AIがビジネスコンテキストを理解し、より正確でブランドに合った結果を生成します。
  </Step>
  <Step title="運用開始">
    - **オムニチャネル**: AIチャットボットを有効にして、24時間365日の自動顧客応答を開始。
    - **Content Hub**: 生成されたアセットを確認し、Content ImproverやRephraseツールで調整して公開。
  </Step>
</Steps>

<Tip>
  今日最も緊急な課題を解決する製品から始めましょう。顧客メッセージの対応に追われているなら、オムニチャネルから。コンテンツの継続的な制作に苦労しているなら、Content Hubから始めてください。
</Tip>

## 無料プランの内容

| 機能 | 無料プラン |
| :-- | :-- |
| 月間メッセージ数 | 200 |
| トレーニングファイル | 3 |
| 統合受信トレイ | 1 |
| 24時間365日自動応答 | 含まれています |
| 自然言語理解 | 含まれています |
