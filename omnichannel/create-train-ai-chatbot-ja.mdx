**パート3：AIチャットボットの作成とトレーニング方法**

### **1：チャットボットを作成**

ステップ1：**_Jinmi Agent_** にアクセス → **_Create a New Assistant_** を選択

ステップ2：情報を入力：

1. チャットボット名を入力（_例：商品相談、よくある質問への回答、24時間365日カスタマーサポート用チャットボット_）
2. チャットボットの役割を説明
3. 主要製品（_例：Jinmi chatbot AI_）
4. ニーズに合わせて機能を選択
5. メッセージ解決時間を設定
6. **_Create_** をクリック

### **2. チャットボットをトレーニング**

- FAQを作成：よくある質問と事前回答のリスト

チャットボットはこれらの質問と回答のペアを使用してお客様に自動応答し、スタッフの時間を節約します。

ステップ1：チャットボット作成後 => **_FAQs_** をクリック => **_Create new FAQ_** をクリック

ステップ2：FAQを追加：

1. お客様がよく聞く質問を入力（_例：この製品には保証がありますか？_）
2. 回答を入力
3. この質問を使用するチャットボットを選択
4. 画像を追加（あれば）
5. **_Create_** をクリック

- チャットボットにドキュメントを添付

Jinmiドキュメントは、チャットボットが読んで学習するための情報、記事、ガイドを保存する場所です。

チャットボットはこれらのドキュメントを使用して、お客様により正確に回答します。

ステップ1：**_Jinmi Agent_** で => **_Documents_** を選択

\

ステップ2：**_Create a new document_** を選択

ステップ3：情報を入力

1. ドキュメント名を入力（_例：Omniユーザーガイド_）
2. ドキュメントタイプを選択
3. ドキュメントリンクを貼り付け
4. チャットボットを選択
5. **_保存_** をクリック

### **3. チャットボットをチャネルに接続**

ステップ1：**_Jinmi Agent_** を選択 => **_Inboxes_** をクリック

ステップ2：**_Connect a new inbox_** を選択

ステップ3：接続したいチャネルを選択（Facebook/Zalo/Website/WhatsApp/Instagram...）

ステップ4：**_Create_** をクリックして受信トレイの接続を完了

この時点で、チャットボットは以下が可能になります：

- お客様への自動応答
- ドキュメントに基づくコンテンツの理解
- 多言語サポート

**注意：** チャットボットのドキュメントは1つの言語で作成するだけで十分です。お客様が別の言語でメッセージを送った場合でも、チャットボットは理解し、お客様の言語で応答します。

**4. チャットボットのテスト**

- 方法1：Jinmi Omnichannelで直接チャットボットをテスト

ステップ1：**_Omnichannel_** にアクセス → **_Conversations_** を選択
ステップ2：接続したチャネルを選択（LINE/Facebook/...）

ステップ3：会話リストで：既存のチャットを選択してボットが動作しているか確認

- 方法2：お客様としてチャット：個人アカウントを使用 → ページ/LINEにメッセージを送信

トレーニングされたコンテンツに基づいてチャットボットが自動応答するのが確認できます。ボットの応答が正確でない場合は、FAQやドキュメントを追加できます。
