---
title: "Webサイトの統合方法"
description: "Jinmiライブチャットウィジェットをあなたのウェブサイトに追加し、サイト訪問者からの問い合わせメッセージをオムニチャネルの統合受信トレイへ自動転送する設定方法をステップバイステップで解説します。HTMLコードの埋め込み手順も含む完全なセットアップガイドです。"
---

ステップ1：**設定** を選択 => **受信トレイ** をクリック => **チャネル接続を追加**

\

ステップ2：**Website** を選択 => 情報を入力 => **受信トレイを作成** をクリック

\

ステップ3：ファンページのサポートエージェントを選択 => **エージェントを追加** をクリック

\
\
\
\

ステップ4：接続成功の通知が表示され、会話に移動できます
