---
title: "Webサイトの統合方法"
description: "Jinmiライブチャットウィジェットをウェブサイトに追加し、訪問者のメッセージをオムニチャネル受信トレイに自動転送する設定手順。"
---

ステップ1：**設定** を選択 => **受信トレイ** をクリック => **チャネル接続を追加**

\

ステップ2：**Website** を選択 => 情報を入力 => **受信トレイを作成** をクリック

\

ステップ3：ファンページのサポートエージェントを選択 => **エージェントを追加** をクリック

\
\
\
\

ステップ4：接続成功の通知が表示され、会話に移動できます
