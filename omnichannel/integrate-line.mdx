---
title: "How to integrate LINE"
---

### **1. Create a LINE Official Account**

Step 1: Select **_Create a LINE Official Account_**

Step 2: Enter the information → Select **_Continue_**

Step 3: Review the information → Select **_Submit_**

Step 4: When **_Application Complete_** is displayed → select **_Go to LINE Official Account Manager_**

### **2. Activate Messaging API**

Step 1: The main screen will show Welcome => select **_Next_** => then select **_Home Screen_** to go to the home page.

Step 2: Select **_Settings_** → Select **_Messaging API_**

Step 3: Click **_Enable Messaging API_**

Step 3: Enter **_Account Name_** → Select **_Agree_**

Step 4: Enter the required information → Select **_OK_**

### **3. Get connection information**

Step 1: Access **_LINE Developers Console_**

Step 2: On the **_Provider_** screen, select the connection channel

Step 3: On the connection channel screen, select **_Messaging API_**

Step 3: Scroll down and select **_Issue_**

### **4. Connect LINE to Jinmi**

Step 1: Go to **_Jinmi Omnichannel_** → Select **_Settings_** → **_Inboxes_**

Step 2: Select **Add Inbox**

**Step 3:** Select **_LINE_** and enter the channel information

Go back to the **_LINE_** screen **→** Copy each item: **_Channel ID, Channel Secret, Token_** to enter the channel information (LINE Channel ID, LINE Channel Secret, LINE Channel Token)

Step 3: Select **_Create LINE Channel_**

Step 4: Add **_Agents_** → Select **_Add Agents_**

Step 5: Copy the **_Webhook URL_** from the Jinmi system

Step 6: Go back to LINE → Paste the **_Webhook URL_** → Select **_Save_**

Step 7: In **_Settings_** → Select **_Response settings_** → Enable **_Chat, Webhook_**

Step 8: LINE has now been successfully connected. Go back to Omnichannel → select **_More Settings_**

You have now completed connecting LINE to Omnichannel and can start using it. Agents can now chat with customers via LINE.
